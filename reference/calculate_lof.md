# calculate_lof

Calculates Local Outlier Factor (lof) for each time series

## Usage

``` r
calculate_lof(this_origvalues_distances)
```

## Arguments

- this_origvalues_distances:

  Dist object for the time series

## Value

Data frame with a feature value per subject.
