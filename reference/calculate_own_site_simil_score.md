# calculate_own_site_simil_score

Calculates the own site similarity feature for each time series.

## Usage

``` r
calculate_own_site_simil_score(this_origvalues_distances, subjects)
```

## Arguments

- this_origvalues_distances:

  Dist object for the time series

- subjects:

  Data frame with one row per study subject.

## Value

Data frame with a feature value per subject.
