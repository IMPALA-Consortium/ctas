# auroc

Fast function for calculating ROC AUC (original code by Miron Kursa
https://mbq.me).

## Usage

``` r
auroc(score, bool)
```

## Arguments

- score:

  Score for ranking the observations.

- bool:

  Boolean giving the observation class.

## Value

ROC AUC value
