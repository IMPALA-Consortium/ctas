# calculate_principal_components

Calculate the first two principal components. These are used to
visualize the relative distance between the time series/subjects.

## Usage

``` r
calculate_principal_components(this_timeseries_wide)
```

## Arguments

- this_timeseries_wide:

  Wide-format data frame with data to calculate the principal components
  for.

## Value

Data frame with the principal components.
